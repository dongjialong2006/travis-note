Github设置

Github操作

新建Gitbook电子书项目，并执行以下操作：

  • 新建分支gh-pages // 该分支专门用于Travis CI发布使用
  • 进入项目的setting，设置域名(非必须)
  • 设置token
  • 等待master分支电子书数据更新
